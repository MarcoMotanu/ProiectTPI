Despre Azure Cloud

Azure este o platformă cloud extinsă oferită de Microsoft. Ea furnizează o gamă variată de servicii pentru a vă ajuta să construiți, să implementați și să gestionați aplicații în cloud.

Servicii Azure

  • Compute (VM-uri, Containere)
  • Stocare (Blob, Tabele, Fișiere)
  • Baze de date (SQL Database, Cosmos DB)
  • Rețele (Virtual Networks, Load Balancers)
